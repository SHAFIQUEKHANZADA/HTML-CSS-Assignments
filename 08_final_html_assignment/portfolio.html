Robiz: is a comprehensive e-commerce platform designed to streamline online shopping experiences. Built with Next.js, TypeScript, and a rich array of UI libraries including Tailwind CSS, Shadcn UI, DevUI, Desy UI, and Material UI, Robiz offers a sleek and responsive interface. It leverages Redux Toolkit and redux-persist for efficient state management, and react-toastify for seamless user notifications. The app features a full-stack architecture that ensures smooth product browsing, secure checkout processes, and efficient order management. Its design prioritizes user experience and responsiveness across all devices, making online shopping intuitive and enjoyable.
